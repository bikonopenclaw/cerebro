# Boletos e malote bancário

Pasta operacional para desenvolver a habilidade do Darth Vader de gerar arquivos de remessa/malote para registro de boletos.

Estrutura:
- `exemplos/`: boletos reais ou modelos enviados pelo Hebert para referência
- `remessas/`: arquivos CNAB gerados futuramente
- `retornos/`: arquivos retorno do banco
- `tratados/`: extrações, CSVs e bases normalizadas

Atenção: dados bancários e de clientes devem ser tratados como sensíveis.
