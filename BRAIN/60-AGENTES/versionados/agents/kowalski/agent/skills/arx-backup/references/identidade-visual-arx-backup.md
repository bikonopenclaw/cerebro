## Direção visual

Relatórios ARX Backup devem usar identidade visual própria do produto ARX Backup.

Não usar marca, logo, assinatura, rodapé, e-mail, timbrado ou identidade visual de terceiros em documentos ARX Backup, salvo se o usuário pedir explicitamente naquele documento específico.

## Paleta operacional derivada das logos

- Preto/chumbo de fundo: `#05080c`
- Azul petróleo: `#0c3f5f`
- Azul ARX/ciano: `#28a9e8`
- Azul claro de destaque: `#6ec6ff`
- Prata/cinza claro: `#d8e1e8`
- Branco: `#ffffff`
- Verde status OK: `#34c759`

## Regras

- Relatórios para cliente devem usar o nome do produto como `ARX Backup`.
- Não citar `Cove` no documento do cliente.
- Não citar marca de terceiros no documento ARX Backup, nem como assinatura.
- Não usar logo, e-mail ou timbrado de terceiros.
- Não incluir seção de recomendações. Usar `Pontos de acompanhamento` quando necessário.
- Horários em GMT-3.
- Preferir relatório em 1 página quando for report mensal executivo.
