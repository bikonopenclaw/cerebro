# Pesquisa empresarial

## Objetivo

Sustentar estratégia e copy com evidências atuais sem imitar concorrentes ou violar direitos.

## Ferramentas

Usar:

1. Web Search do OpenClaw para descoberta;
2. Exa como mecanismo preferencial de pesquisa semântica, quando configurado;
3. Firecrawl somente para extrair páginas cuja informação necessária não esteja acessível pela busca;
4. documentos internos fornecidos pela empresa como fontes primárias do negócio.

Se a fonte aprovada não estiver disponível, parar a pesquisa naquele ponto e apresentar as opções. Não trocar Exa, Firecrawl, busca configurada ou fonte do usuário sem decisão explícita.

## Ordem de fontes

Priorizar:

1. documentação e canais oficiais;
2. dados públicos primários;
3. estudos e relatórios metodologicamente claros;
4. imprensa reconhecida;
5. fontes secundárias somente com confirmação.

Para regras, preços, produtos, recursos, leis ou dados que possam mudar, verificar no momento da produção.

## Registro

Guardar em `research.md`:

| Campo | Conteúdo |
|---|---|
| URL ou arquivo | Fonte exata |
| Data de acesso | Data e fuso |
| Afirmação | O que a fonte sustenta |
| Uso | Estratégia, copy, referência ou compliance |
| Limitação | Incerteza, recorte ou possível desatualização |
| Direitos | Condições conhecidas de reutilização |

Não guardar uma conclusão sem a fonte que a sustenta.

## Concorrentes e tendências

- Usar concorrentes para mapear território, não para copiar execução.
- Não reproduzir composição, personagem, slogan ou roteiro distintivo.
- Tratar tendência como sinal, não como obrigação.
- Rejeitar uma tendência incompatível com marca, público ou compliance.
- Não copiar trechos extensos, imagens ou vídeos protegidos.
- Produzir uma direção original a partir dos aprendizados.

## Saída

Entregar:

- fatos verificados;
- sinais de mercado;
- lacunas;
- implicações editoriais;
- oportunidades;
- riscos;
- fontes.
