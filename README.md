## Arquitetura

O Brain não é agente. É o repositório vivo de conhecimento da instância.

O Puppet Master administra o Brain, consolida conhecimento e aciona Darth Vader ou Kowalski quando a tarefa exigir especialidade.

A consolidação diária roda em modo silencioso para preservar contexto sem atrapalhar execução.
