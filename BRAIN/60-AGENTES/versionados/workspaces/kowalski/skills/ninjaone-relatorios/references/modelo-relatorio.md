# Modelo base de relatório NinjaOne Bikon

## Estrutura recomendada

1. Capa
   - Logo Bikon
   - Cliente ou visão geral
   - Período de referência
   - Data de emissão

2. Sumário executivo
   - 1 parágrafo curto com situação geral
   - 3 a 5 bullets de atenção

3. Indicadores
   - Organizações, quando relatório geral
   - Dispositivos monitorados
   - Dispositivos offline
   - Alertas ativos
   - Alertas recorrentes, se houver histórico

4. Riscos e prioridades
   - Crítico
   - Atenção
   - Informativo

5. Tabelas técnicas
   - Organização
   - Dispositivo
   - Alerta
   - Último contato
   - Ação recomendada

6. Recomendações
   - O que fazer
   - Por que importa
   - Próximo passo
